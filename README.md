# 🤖 Beginner-Friendly AI Chatbot

This is a simple AI chatbot website made with **Python (Flask)** and **OpenAI’s ChatGPT (GPT-3.5)**. It has a clean and easy-to-use design, just like ChatGPT. You can type your question, and the bot will reply back.

---

## 🧠 What Can It Do?

- Lets you type messages and get smart replies from ChatGPT
- Looks like a real AI assistant in dark mode
- Runs on your own computer (locally)
- Easy to understand code (perfect for beginners)

---

## 🖼️ What It Looks Like

The chatbot has a dark theme with a clean and simple chat layout.
